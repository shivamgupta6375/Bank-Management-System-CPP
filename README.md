# Bank Management System
A console-based Bank Management System developed in C++. This project simulates basic banking operations through a user friendly terminal interface.

## Features 
- User Registration 
- User Login Authentication
- Deposit Money
- Withdraw Money
- Check Account Balance
- Change Password
- View Account Details
- Secure Logout

## Technologies Uses
- C++
- Visual Studio Code
- Standard Template Library (STL)

## How to Run
1. Clone the repository.
2. Open the project in Visual Studio Code.
3. Compile the source file:
   ```bash
   g++ main.cpp -o bank
   ```
4. Run the program:
   ```bash
   ./bank
   ```
